MetaMapp : Mapping of Metabolites by Chemical and Biochemical relationships

Generate MetaMapp files for visualizing metabolomics networks.

First three columns in the input file should be PubChem ID, KEGG ID and Name of Compounds, then followed by pair of columns for pvalue and fold-changes for each statistical comparison. Please use the example file as a templete. It is recommended to use median to calcualte the fold-changes, as median are not sensitive to outliers. For getting KEGG or PubChem IDs for metabolites names, use ChemIDConvert tool. For statistical analysis, use MetaBox software. A pvalue of 0.05 will be used to identify the significantly altered metabolites. Output files are a Cytoscape network in .SIF format and a node attribute file. Cytoscape can be downloaded from here Cytoscape.Org

Generate Node attribute files for Cytocape visualization

First row should be header describing class names of each sample and the first column should contain PubChem CIDs. Eeach row should be peak intensities of metabolites across samples. This tool would calculate a T-Test (two sided and equal variance) between the two groups for each metabolite and return p-values and fold-change (median of class 1/median of class 2) . Minimam number of samples per group should be three. And both classes should have equal number of samples. Example Data File
